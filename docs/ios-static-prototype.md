# iOS 静态原型

`v0.2.0` 的目标是先跑通一个不接后端、不接真实 AI 的 SwiftUI 静态原型。

## 当前页面

- 今日学习：展示任务状态、完成进度、5 句精听、句中关键单词、日常语法结构和词块。
- 素材主题：按学习阶段筛选 mock 主题，点进详情可看到情景剧/歌词/新闻稿/文章摘录等 mock 内容，并可模拟生成今日 15 分钟练习。
- 跟读练习：展示目标句、录音占位、mock 口语评分、转写和建议；跟读完成后依次展示听写、默写。
- 学习复盘：展示连续学习、累计分钟、可迁移表达、今日词块和明日重点。

## 运行方式

需要安装完整 Xcode。

1. 打开 `EnglishLearningApp.xcodeproj`。
2. 选择 `EnglishLearningApp` scheme。
3. 选择 iPhone 模拟器。
4. 点击 Run。

已在 Xcode 26.6 中使用 iPhone 17 Pro 模拟器成功运行静态原型。
